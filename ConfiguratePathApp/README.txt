
  Configuration Application for the MOPLUGS Project.
  
 This is special application which could help to setup plugin and script path for Autodesk MotionBuilder.
 
  This application operate with the current user environment variables
  MOTIONBUILDER_PLUGIN_PATH
  MOTIONBUILDER_PYTHON_STARTUP
